# Culebrita en Python con Interfaz Gráfica

Este es un juego simple de la culebrita implementado en Python utilizando la biblioteca gráfica `turtle`. En este juego, controlarás una culebra que se moverá en la pantalla, tratando de comer la comida para crecer y obtener puntos.

## Requisitos

Asegúrate de tener Python instalado en tu sistema. Este código fue probado con Python 3.

## Ejecución

Ejecuta el script `snake_game.py` para comenzar el juego.

```bash
python snake_game.py
```

Controla la culebra con las teclas de flecha (arriba, abajo, izquierda, derecha). El objetivo es comer la comida (puntos rojos) sin chocar contra los bordes de la pantalla o el cuerpo de la culebra.

## Reglas del Juego

1. La culebra crece cada vez que come la comida.
2. Evita chocar contra los bordes de la pantalla o contra el cuerpo de la culebra.
3. La puntuación se incrementa por cada comida consumida.
4. La mejor puntuación se muestra en la parte superior de la ventana.

## Características

- La culebra se mueve con una velocidad gradualmente incrementada.
- La posición de la comida es aleatoria cada vez que es consumida.
- El juego tiene detección de colisiones para los bordes y el cuerpo de la culebra.

## Personalización

Siéntete libre de modificar y expandir el código según tus preferencias. Puedes ajustar el tamaño de la pantalla, la velocidad de la culebra y otras características para adaptar el juego a tus necesidades.

¡Disfruta jugando a la culebrita!
